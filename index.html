/ 04 — OUTBOUND PLAY

THE PITCH. THE WEDGE.

Don't sell installs. Sell the BMS Health Check. £1,500 audit. Low commitment. 70% of audits convert to a retainer or install.

O-01
Build target list — 200 contacts
LinkedIn Sales Nav. Filter: Facilities Director / Chief Engineer / Head of Estates · London + M25 · Hotels (4–5★) / Hospitals / REITs / Data centres. Export to Airtable.
WK 2P1
O-02
Write the wedge message
3 lines max. Hook: "Most BMS systems in London hotels are 15-30% over-spending on energy." Offer: free 30-min review or £1.5k Health Check. Proof: Hilton-grade portfolio.
WK 2P1
O-03
Send 50 LinkedIn DMs / week
10/day, Mon–Fri. Track in Airtable. Expected reply rate 8–12%. Expected booked-call rate 2–4%. Target: 2+ calls/wk by week 4.
WK 3+P1
O-04
AI outbound caller live
Build a Vapi.ai or Bland.ai agent (~£0.07/min). Calls cold lists Mon–Fri 10:00–16:00. Script: qualify on BMS brand, age, fault history, decision-maker. Books a 15-min call with Bafana.
WK 4P2
O-05
Build the BMS Health Check report template
12-page branded PDF. Sections: site overview · controller audit · sensor calibration · control loops · energy waste · prioritised remediation. Use template for every audit — 5-day turnaround standard.
WK 3P1
O-06
Add /health-check page to site
Scope, fixed pricing (£1.5k / £2.5k / £3.5k by sqft), sample report download, single CTA: "Book a site survey". Becomes primary conversion path.
WK 3P1
O-07
Upsell ladder defined
Health Check (£1.5–3k) → remediation work (£5–25k) → monitoring retainer (£750–£2k/mo) → OD Pulse AI dashboard (+£400/mo). Every quote shows the ladder.
